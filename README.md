# Sophia
## Assistant integrated in Telegram

**A little about Sophia:**

- operates on the basis of **Telegram API**
- full asynchronous assistant
- integrated database is **MySQL** (asyncio model - AioMySQL)
- using OpenAI products, can be dynamic
